# uma-sports-oracle

## Contracts

- `UmaSportsOracle` is a contract that allows users to create and resolve sports games and markets.

## Modules

- `Auth` is a generic autorization module with multiple owners.

## Libraries

- `ScoreDecoderLib` is an internal library for decoding scores according to UMIP-183

- `PayoutLib` is an internal library for constructing CTF payouts for different market types
